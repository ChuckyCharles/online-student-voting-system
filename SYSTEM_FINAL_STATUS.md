## ⚡ Quick Commands

### Start Backend
```powershell
cd backend
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

### Start Frontend
```powershell
cd frontend
npm run dev
```

### Reset Database
```powershell
cd backend
rm voting_system.db
python setup_machakos_election.py
```
